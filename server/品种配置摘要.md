## 后台管理接口

### 基础URL
```
http://localhost:3001/api/symbol
```

### 主要接口

| 方法 | 路径 | 说明 | 权限 |
|-----|------|------|------|
| GET | /list | 获取所有品种 | - |
| GET | /active | 获取活跃品种（前端用） | - |
| GET | /detail/:id | 获取品种详情 | - |
| POST | /add | 添加新品种 | 管理员 |
| PUT | /update/:id | 更新品种 | 管理员 |
| DELETE | /delete/:id | 删除品种 | 管理员 |
| PUT | /toggle/:id | 启用/禁用 | 管理员 |
| PUT | /sort | 批量排序 | 管理员 |
| GET | /categories | 获取分类列表 | - |

## 添加新品种流程

### 1. 测试数海代码可用性

```bash
node test-single-symbol.js <数海代码>
```

示例：
```bash
node test-single-symbol.js CEESA0
```

### 2. 通过API添加品种

```bash
curl -X POST http://localhost:3001/api/symbol/add \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "code": "ES",
    "name": "小标普",
    "shuhaiCode": "CEESA0",
    "market": "CE",
    "category": "指数",
    "pricePrecision": 2,
    "sort": 6
  }'
```

### 3. 前端自动同步

前端调用 `/api/symbol/active` 接口获取最新品种列表

## 前端对接说明

### 获取品种列表

```javascript
// 获取活跃品种（前端使用）
const response = await fetch('/api/symbol/active');
const symbols = await response.json();

// symbols格式:
// [
//   { code: 'DAX', name: '德指', shuhaiCode: 'CEDAXA0', market: 'CE', category: '指数', pricePrecision: 2 },
//   ...
// ]
```

### 获取实时行情

```javascript
// 方式1: 批量获取所有品种
const response = await fetch('/api/market/batch');
const data = await response.json();

// 方式2: 获取单个品种
const response = await fetch('/api/market/quote?code=DAX');
const data = await response.json();
```

## 注意事项

1. **前端无需修改**: 品种列表通过API动态获取
2. **后台可添加**: 管理员可通过后台管理添加新品种
3. **权限验证**: 添加品种前必须验证数海代码是否可用
4. **自动同步**: 前端刷新后自动获取最新品种列表
5. **黄金品种**: 需要购买数海黄金市场权限后才能添加
