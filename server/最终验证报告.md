## 后台管理接口

### 基础信息

- **基础URL**: `http://localhost:3001/api/symbol`
- **认证方式**: Bearer Token
- **响应格式**: JSON

### 主要接口

| 方法 | 路径 | 功能 | 权限 |
|-----|------|------|------|
| GET | /list | 获取所有品种 | 无 |
| GET | /active | 获取活跃品种（前端用） | 无 |
| GET | /detail/:id | 获取品种详情 | 无 |
| POST | /add | 添加新品种 | 管理员 |
| PUT | /update/:id | 更新品种 | 管理员 |
| DELETE | /delete/:id | 删除品种 | 管理员 |
| PUT | /toggle/:id | 启用/禁用 | 管理员 |
| PUT | /sort | 批量排序 | 管理员 |
| GET | /categories | 获取分类列表 | 无 |

## 系统特性

### 自动化功能

1. **自动更新**: 每30秒从数海API自动更新行情数据
2. **数据缓存**: 5秒缓存，减少API调用次数
3. **错误重试**: API不可用时自动重试连接
4. **批量获取**: 一次请求获取所有品种数据

### 管理功能

1. **动态品种**: 后台可随时添加、删除品种
2. **前端同步**: 前端自动获取最新品种列表
3. **状态管理**: 可启用/禁用品种
4. **分类管理**: 按分类组织品种
5. **排序控制**: 可调整品种显示顺序

## 前端使用说明

### 获取品种列表

```javascript
// 获取活跃品种（推荐）
const response = await fetch('/api/symbol/active');
const symbols = await response.json();

// 返回格式:
// {
//   code: 0,
//   message: 'success',
//   data: [
//     { code: 'DAX', name: '德指', shuhaiCode: 'CEDAXA0', ... },
//     { code: 'HSI', name: '恒指', shuhaiCode: 'HIHHI02', ... },
//     ...
//   ],
//   timestamp: 1740480000000
// }
```

### 获取实时行情

```javascript
// 批量获取所有品种
const response = await fetch('/api/market/batch');
const data = await response.json();

// 单个品种
const response = await fetch('/api/market/quote?code=DAX');
const data = await response.json();
```

```bash
curl -X POST http://localhost:3001/api/symbol/add \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "code": "ES",
    "name": "小标普",
    "shuhaiCode": "CEESA0",
    "market": "CE",
    "category": "指数"
  }'
```
