## 响应式断点

```css
/* 移动端（竖屏手机） */
@media (max-width: 768px) { ... }

/* 平板端 */
@media (min-width: 769px) and (max-width: 1024px) { ... }

/* 桌面端 */
@media (min-width: 1025px) { ... }

/* 横屏手机 */
@media (max-width: 768px) and (orientation: landscape) { ... }
```

## 使用建议

### Tailwind CSS 响应式类
```tsx
/* 网格布局 */
<div className="grid grid-cols-2 md:grid-cols-4 gap-3">

/* 弹窗宽度 */
<Dialog width="90vw" style={{ maxWidth: '600px' }}>

/* 图表高度 */
<ReactECharts style={{ height: '250px' }}>

/* 文本隐藏/显示 */
<span className="hidden md:block">桌面端显示</span>
<span className="md:hidden">移动端显示</span>
```

### 自定义响应式类
```tsx
<div className="dialog-responsive">
<div className="chart-mobile">
<div className="table-container">
```
