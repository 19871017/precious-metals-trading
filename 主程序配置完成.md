## 前端使用

### 1. 获取品种列表

```javascript
import { getMarketSymbols } from '@/services/shuhai-backend.service';

const symbols = await getMarketSymbols();
console.log(symbols); // 6个品种的列表
```

### 2. 获取单个品种行情

```javascript
import { getQuoteBySymbol } from '@/services/shuhai-backend.service';

const goldQuote = await getQuoteBySymbol('GOLD');
const hsiQuote = await getQuoteBySymbol('HSI');
```

## 注意事项

1. **必须重启服务**才能加载新代码
2. **数海API有频率限制**，避免过于频繁请求
3. **批量请求必须至少2个品种**，单个品种返回407
4. **只支持HTTP协议**，不支持HTTPS

## 数据流

```
数海API (ds.cnshuhai.com)
    ↓ HTTP请求
后端代理 (/shuhai/*)
    ↓ 缓存 (5秒)
前端API调用
    ↓ 响应
前端显示
```
