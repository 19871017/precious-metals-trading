### 问题原因
1. 使用了代理服务器地址 `https://api.lxd997.dpdns.org/stock.php`，该服务器需要代理认证(407/406错误)
2. 账号密码不正确：正确密码是 `wu123`，而不是 `wu123456`

### 解决方案
改用官方API地址，并使用正确的账号密码：

| 配置项 | 旧值 | 新值 |
|--------|------|------|
| API地址 | https://api.lxd997.dpdns.org/stock.php | http://ds.cnshuhai.com/stock.php |
| 用户名 | wu123 | wu123 |
| 密码 | wu123456 | wu123 |

### 注意事项

1. **无需签名验证** - 官方API直接使用 `type=stock&u=username&p=password` 即可
2. **批量请求** - 支持用逗号分隔多个品种代码，一次请求获取多个品种
3. **缓存策略** - 代码中已实现5秒缓存，避免频繁请求
4. **更新频率** - 代码中设置为30秒自动更新一次

### 后续优化建议

1. 如需访问更多品种，需要购买数海相应市场的数据权限
2. 可以考虑增加重试机制，提高API调用的可靠性
3. 可以添加API健康检查，在API不可用时自动降级到模拟数据
